Phaser Game

An old-style space shooter game!


ABOUT

This is a browser game, enterely made in JavaScript and a powerful framework. The style is similar to the old space shooter games, that could be played in the coolest game rooms, but easily aviable online!

CONTROLS

Arrow Keys : Move your cool ship
Spacebar: Fire some bullets
R : Restart the game
M : Mute BGM
